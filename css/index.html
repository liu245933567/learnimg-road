# Less

Lessopen in new window （Leaner Style Sheets 的缩写） 是一门向后兼容的 CSS 扩展语言。

# Sass

Sassopen in new window 是一款强化 CSS 的辅助工具，它在 CSS 语法的基础上增加了变量 (variables)、嵌套 (nested rules)、混合 (mixins)、导入 (inline imports) 等高级功能，这些拓展令 CSS 更加强大与优雅。使用 Sass 以及 Sass 的样式库（如 Compass）有助于更好地组织管理样式文件，以及更高效地开发项目。
